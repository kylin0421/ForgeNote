# 高校专业课知识库演示数据

本目录提供一套可直接导入ForgeNote的回归测试数据，用于验证文档解析、检索索引、问答和学习资产生成。数据面向人工智能、计算机与电子信息相关专业的典型课程场景，不作为学术内容准确性基准。

## 数据组成

- 原始课程资料：[人工智能&python_知识点整理.docx](人工智能&python_知识点整理.docx)
- 示例知识库：`人工智能与 Python 专业课知识库`
- 示例播客：[人工智能与 Python 期末复习：从机器学习到知识推理](人工智能与Python专业课知识库-播客.mp3)，时长 14 分 23 秒
- 已验证学习资产：课程讲解、测验、9 张闪卡、思维导图、拓展阅读、代码实验和中文播客

课程资料覆盖六个模块：

| 模块 | 主要内容 |
| --- | --- |
| Python 基础 | 数据类型、流程控制、函数、文件、异常与 NumPy |
| 机器学习 | 线性回归、逻辑分类、KNN、K-means 与朴素贝叶斯 |
| 神经网络 | 全连接网络、CNN、Keras 流程、参数量与卷积计算 |
| 知识表示 | 谓词逻辑、产生式系统、语义网络与框架 |
| 推理 | 正向/反向推理与确定性因子推理 |
| 搜索与优化 | 状态空间、A* 与遗传算法 |

## 建议验证项

1. 导入 DOCX，并等待“已建立检索索引”显示为“是”。
2. 使用来源约束问答检查章节定位与引用。
3. 生成课程讲解、测验、闪卡、思维导图、拓展阅读、代码实验和播客。
4. 检查任务悬浮球中的进度、失败原因和日志。
5. 验证测验作答、闪卡翻转、思维导图展开、代码实验导出与播客播放/导出。
