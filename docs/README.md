- [配置指南](configuration-guide.md)：环境变量、数据库、前端/API、模型/API key、默认模型、TTS 与排错。
- [测试说明](testing.md)：测试范围、测试方法、命令、结果、已知风险。
- [部署与演示指南](deployment-and-demo.md)：环境准备、本地/Docker 运行、演示脚本、截图说明。
- [开源与 AI 工具说明](open-source-and-ai-tools.md)：开源底座、主要依赖、许可证、AI Coding 工具使用说明。

## 最近功能补充

- 学习记录页新增独立“学习曲线”入口，用于展示最近 14 天学习量、学习质量、测验正确率、每日明细和近期学习建议。
- 错题本不再放在左侧来源栏，而是作为学习记录顶部独立入口，便于和学习曲线保持同级但不干扰资料区。
- 学习画像展示改为面向用户的摘要视图，自动压缩后台学习信号、题干和错题记录，突出背景、当前目标、易错点和资源偏好。
- 来源栏将“添加来源”和“搜集资料”调整为标题下方的大按钮，便于用户继续导入网页、文件、音频或文本资料。
- Studio 侧支持更清晰的学习资产入口、测验总结与相似题练习、播客播放器队列、字幕和倍速控制，以及可展开的学习资产展示。
- 对话区支持框选回答后引用继续追问，并提供下一句推荐问题。
- 思维导图侧重可视化树状展示、默认折叠、全屏查看和后续编辑能力；代码实验室中的 notebook 代码内容可编辑，其他资产默认不显示编辑按钮。
- 导出规则已按资产类型收敛：课程学习讲解、错题本、思维导图、代码实验室和播客支持对应导出；拓展阅读和闪卡不导出。
- 模型/API 配置页提供基础默认模型和高级覆盖模型，基础区包含通用文本、Embedding、图片、TTS、STT，图片模型可配置 DashScope/Qwen `qwen-image`。
- 播客 episode 已增加 notebook 绑定字段，避免音频生成成功但 Studio 侧查不到当前学习记录的播客。
